この物件専用の LINE 公式アカウントを紐付けます。
複数の Bot を登録すると、無料枠 (200通/月) を使い切った際に自動で切り替えられます。

  1. https://developers.line.biz/console/ を開く
  2. 「長浜清掃G通知」(or 新規) プロバイダー → 「新規チャネル作成」
  3. 「Messaging API」を選択
  4. チャネル名: 例「○○物件 通知Bot #1」と入力して作成
  5. 「Messaging API設定」タブ → 「チャネルアクセストークン」で発行
  6. トークンをコピーして Bot #1 のフォームに貼り付け
  7. LINE グループを作成 → 作成した Bot を招待
  8. グループ内で Bot にメンション → サーバーログで Group ID を確認
  9. Group ID を Bot #1 のフォームに貼り付け → 保存
  10. 2つ目の Bot は同様の手順で Bot #2 として登録
