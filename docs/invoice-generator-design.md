# 請求書自動生成 — アプリ設計書

## 概要
清掃スタッフ（業務委託）の月次請求書を、シフト実績+コインランドリー+交通費から自動計算。
スタッフが内容確認→編集→送信ボタンで完了する仕組み。

## フロー

```
月末（自動トリガー）
  ↓
シフト実績集計（スタッフ別）
  + コインランドリー費集計
  + 交通費計算
  + 特別手当
  ↓
請求書データ生成（スプレッドシートに書き込み）
  ↓
スタッフにメール送信
  「○月分の請求書が準備できました。内容をご確認ください」
  → 確認ページURL付き
  ↓
スタッフが確認ページを開く
  - 明細を確認
  - 必要に応じて編集（交通費実費の入力、メモ追加等）
  - 「この内容で請求する」ボタン
  ↓
請求書PDF自動生成
  ↓
オーナーにメール送信（PDF添付）
  ↓
ステータス: 確定
```

## 請求書の項目

| 項目 | 計算方法 |
|------|----------|
| 基本報酬 | 出勤回数 × 単価（スタッフマスタから） |
| コインランドリー費 | 月間合計（コインランドリー記録から） |
| 交通費 | 固定額 × 出勤日数 or スタッフが実費入力 |
| 特別手当 | 手動設定（繁忙期手当等） |
| 消費税 | 業務委託なのでインボイス対応要確認 |
| 合計 | 上記合計 |

## スタッフ確認画面（モバイル最適化）

```
━━━ 2026年3月分 請求書 ━━━

山田太郎 様

■ 基本報酬
  出勤回数: 12回
  単価: 5,000円
  小計: 60,000円

■ コインランドリー費
  利用回数: 8回
  小計: 4,800円

■ 交通費
  12回 × 500円 = 6,000円
  [実費に変更する]

■ 特別手当
  なし

━━━━━━━━━━━━━━━━━
合計: 70,800円

[内容を修正する]  [この内容で請求する]
```

## 技術的注意点
- インボイス制度対応（適格請求書発行事業者番号の記載）
- PDF生成はGASのHtmlService + Google Docs変換 or 外部ライブラリ
- スタッフ確認ページはGAS Web Appとして公開（スタッフ別URL or 認証）
